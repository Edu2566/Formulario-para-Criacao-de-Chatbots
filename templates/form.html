Mensagens e subopções

Defina as mensagens principais e crie subopções (filhas) infinitamente. Ex.: “Digite 1 para fazer X” pode ter “Digite A para Y”, e assim por diante.

Como organizar:

  • Use “Nova mensagem principal” para criar um passo inicial.
  • Clique em “+ Subopção” dentro de uma mensagem para criar ramificações.
  • Use “Remover” para excluir o bloco atual (mantemos pelo menos uma raiz).
  • Colapse nós para ter uma visão mais limpa quando houver muitas subopções.
